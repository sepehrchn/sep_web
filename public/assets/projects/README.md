## Image Requirements

- **Aspect Ratio:** 16:10 (recommended: 1600×1000px or 1280×800px)
- **Format:** JPG or PNG
- **Optimization:** Compress images for web (aim for <200KB per image)
- **Content:** Real project screenshots, UI mockups, or product visuals

## File Naming

Each project has 3 images:
- `{project-name}-1.jpg` — Primary screenshot (hero/landing page)
- `{project-name}-2.jpg` — Secondary feature or UI detail
- `{project-name}-3.jpg` — Third view or mobile/responsive view
