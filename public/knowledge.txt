== هویت ==

تحصیلات: کارشناسی هوش مصنوعی (گرایش یادگیری ماشین و پردازش زبان طبیعی) از دانشگاه APU
وضعیت همکاری: آماده برای شروع همکاری تمام‌وقت یا پروژه‌ای (قراردادی)

== توانایی‌های اصلی ==
۱. توسعه وب فول‌استک — ساخت سیستم‌های وب ماژولار، سریع و type-safe با React، Next.js و TypeScript
۲. هوش مصنوعی و پردازش زبان طبیعی — آموزش مدل‌های BERT، طبقه‌بندی‌کننده‌های سفارشی، و orchestration مدل‌های زبانی بزرگ (LLM) با pipeline های RAG ترکیبی
۳. اتوماسیون فرآیند و گردش‌کار — ساخت pipeline های خودکار و مقاوم با n8n، Zapier، Make و فناوری‌های edge سرورلس (Cloudflare Workers)

== پشته فناوری (Tech Stack) ==
هوش مصنوعی/یادگیری ماشین: PyTorch، Transformers/BERT، Gemini API، OpenAI API، RAG / بازیابی ترکیبی (Hybrid Retrieval)، ElevenLabs (تولید صدا)
فول‌استک: Next.js، React، TypeScript، Node.js، FastAPI، PostgreSQL، Prisma، Tailwind CSS، Framer Motion، Server Actions
اتوماسیون و زیرساخت: n8n، Zapier، Make، REST APIs، Webhooks، Cloudflare Workers، D1، KV، CI/CD (مقدماتی)، Git، Figma، Docker، Vercel

== زبان‌ها ==
فارسی (مادری)، انگلیسی (سطح C1)

== نمونه‌کارها و پروژه‌ها ==

۱. پلتفرم فروشگاهی شاه‌مغز
فروشگاه آنلاین خشکبار و آجیل با دیتابیس شامل بیش از ۱۳ مدل داده، رابط کاربری کاملاً بومی‌سازی‌شده و راست‌چین (RTL)، همراه با داشبورد مدیریتی برای کنترل سفارش‌ها و محصولات.
دمو: https://shah-maghz.vercel.app/

۲. پورتال تجاری آریانا
پلتفرم تجارت B2B صادراتی با پشتیبانی دوزبانه (فارسی/انگلیسی)، چیدمان تطبیقی RTL/LTR، و چت‌بات ترکیبی که دانش محلی کسب‌وکار را با یک مدل زبانی بزرگ (LLM) ترکیب می‌کند.
دمو: https://arianasepehr.vercel.app/

۳. وب‌سایت استودیو FORMA
وب‌سایت با هیرو سه‌بعدی تعاملی (WebGL)، پشتیبانی از ۴ زبان، و ویجت چت هوش مصنوعی اختصاصی طراحی‌شده برای برند.
دمو: https://adart-alpha.vercel.app/

۴. ربات دستیار هوش مصنوعی در تلگرام
دستیار مبتنی بر Gemini API، اجرا‌شده روی Cloudflare Edge برای پاسخ‌دهی سریع، با قابلیت پاسخ صوتی (ElevenLabs) و تشخیص خودکار زبان کاربر.
دمو: به‌زودی تکمیل می‌شود

۵. خودکارسازی ارتباط با مشتری
Pipeline خودکار برای مدیریت و پاسخ‌دهی به ارتباطات مشتریان با n8n، شامل مرحله بازبینی انسانی (human-in-the-loop) و محدودسازی نرخ درخواست‌ها (rate limiting).
دمو: به‌زودی تکمیل می‌شود
